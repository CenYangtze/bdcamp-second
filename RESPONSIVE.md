# 响应式布局说明

本项目已完成响应式布局优化，支持多种设备尺寸。

## 断点设置

根据 Arco Design Grid 系统和常见设备尺寸：

| 断点 | 设备类型 | 屏幕宽度 | 说明 |
|------|---------|---------|------|
| `xs` | 手机 | < 576px | 超小屏幕 |
| `sm` | 手机/平板 | ≥ 576px | 小屏幕 |
| `md` | 平板 | ≥ 768px | 中等屏幕 |
| `lg` | 桌面 | ≥ 992px | 大屏幕 |
| `xl` | 大屏桌面 | ≥ 1200px | 超大屏幕 |

## 布局适配

### 1. 导航栏 (Navigation)

- **桌面** (> 768px)：Logo + 搜索框 + 购物车横向排列
- **平板** (≤ 768px)：缩小间距，搜索框宽度适配
- **手机** (≤ 576px)：Logo 和购物车在上，搜索框换行到下方

### 2. 商品列表 (ProductList)

#### 筛选面板
- **桌面** (≥ 992px)：左侧固定筛选栏（6列）
- **平板** (768px - 991px)：左侧筛选栏（8列）
- **手机** (< 768px)：筛选栏占满整行（24列），不固定

#### 商品卡片网格
- **超大屏** (≥ 1200px)：每行 4 个商品（6列 × 4）
- **桌面** (992px - 1199px)：每行 3 个商品（8列 × 3）
- **平板** (768px - 991px)：每行 2 个商品（12列 × 2）
- **手机** (< 768px)：每行 2 个商品（12列 × 2）

### 3. 商品详情 (ProductDetail)

#### 布局
- **桌面** (≥ 768px)：左侧图片（10列）+ 右侧信息（14列）
- **手机** (< 768px)：上下堆叠，图片和信息各占满整行（24列）

#### 操作按钮
- **桌面**：按钮横向排列
- **手机**：按钮纵向堆叠，占满宽度

### 4. 商品卡片 (ProductCard)

- **桌面**：图片高度 200px，字体标准大小
- **平板**：图片高度 180px，略小字体
- **手机**：图片高度 160px，按钮纵向排列

## CSS 媒体查询

所有响应式样式使用标准 CSS 媒体查询：

```css
/* 平板设备 */
@media (max-width: 992px) {
  /* 样式 */
}

/* 手机设备 */
@media (max-width: 576px) {
  /* 样式 */
}
```

## 测试建议

使用浏览器开发者工具测试以下设备：

1. **手机**：iPhone SE (375px), iPhone 12 Pro (390px)
2. **平板**：iPad (768px), iPad Pro (1024px)
3. **桌面**：常规桌面 (1440px), 高分辨率 (1920px)
